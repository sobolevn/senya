# senya

Minimalistic Hyper theme inspired by old-fashioned hobbies.

![main](https://raw.githubusercontent.com/sobolevn/senya/master/media/main.png)

Core features:
- Minimalistic
- Removes extra UI
- Borderless
- High contrast for the needed parts
- Comfortable colors
